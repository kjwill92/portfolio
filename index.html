HilaryBakes.com - Personal Project

React | Redux | HTML | CSS | Javascript | Node | Express | PostgreSQL | Massive | AWS S3

I created a web application to showcase my sister's talent in the Culinary field. Visitors to the site can view some of her creations as well as make a request for something themselves. On her end, she has the ability to add new products which get stored on the database. She can also view all the order requests. I hosted my website with Digital Ocean/Heroku.

I used JavaScript in the React framework to create functionally. This allows the users to view content, and the admin to edit that content, and delete the content as they see fit.

Node and Express handle the back-end RESTful calls to a cloud-based PostreSQL database.

https://github.com/kjwill92/personalProject | www.hilarybakes.com

U-Tube - Group Project

You might have guessed: a new and improved media sharing platform! My team set out to create a video sharing service like Youtube but without many of the frills that can make Youtube obnoxious. A user can view, like, and comment on videos as well as create an account through which they can upload and manage their own content.

Videos are stored using Amazon S3 file storage and a smooth and secure login/registration is facilitated by the use of 0Auth. Team collaboration and version control was accomplished using Github.

React | HTML5 | CSS3 | NodeJS | Axios | Express | Massive | Heroku | PostgreSQL | 0Auth | Amazon S3 | Dropzone

https://github.com/dm-youtube-clone/youtube-clone
